# Автоматизация тестирования веб‑сервиса «Путешествие дня»

Автоматизированные тесты для веб‑приложения, позволяющего оформить покупку тура с оплатой банковской картой.  
Тесты проверяют корректность работы пользовательского интерфейса, взаимодействие с тестовым банковским сервисом и запись результатов в базу данных PostgreSQL.

---

## Начало работы

Ниже приведены инструкции, которые помогут получить копию проекта и запустить его на локальном компьютере.

---

## Prerequisites

Перед запуском необходимо установить:

- Git
- Java 11 (JDK 11)
- Docker и Docker Compose
- Любой современный браузер (Chrome / Firefox)

---

## Установка и запуск

### 1. Клонирование репозитория

git clone <ссылка-на-репозиторий>  
cd <папка-проекта>

### 2. Запуск базы данных PostgreSQL

docker-compose up -d

Проверка:

docker ps

Ожидаемый контейнер:

postgres-db

### 3. Запуск тестируемого сервиса

Поместите файл aqa-shop.jar в корень проекта и выполните:

java -jar aqa-shop.jar

Сервис будет доступен по адресу:

http://localhost:8080

### 4. Запуск автотестов

./gradlew clean test

### 5. Формирование отчёта Allure

./gradlew allureReport  
./gradlew allureServe
